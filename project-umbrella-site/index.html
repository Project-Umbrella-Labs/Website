What we do

Inspired by high‑reliability engineering practices (think Northrop Grumman), we operate like a small engineering firm: design reviews, verification & validation, issue tracking, and proper docs. We also love bold, practical builds—much like UNSW Rocketry and Breaker Industries.

Aerospace

Satellite & ground systems

Tracking, telemetry and SDR. Antenna pointing, RF chain integration, and robust Python/C++ software to automate the pipeline.

Robotics

Mechatronics & control

Motor drivers, motion planning, hardware‑in‑the‑loop testing, and real‑time control on STM32 / RP2040 / Linux SBCs.

Software

High‑reliability code

CI pipelines, code reviews, structured logging, and fault‑tolerant services. From embedded C to Rust and TypeScript.
