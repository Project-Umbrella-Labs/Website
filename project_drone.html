Atmospheric Drone Mk II

/// MK II OVERVIEW

Current Mission Objectives

The Mark II represents a significant leap in our autonomous capabilities. Focusing on high-efficiency flight paths and real-time edge computing, the Mk II frame is optimized for flight times exceeding 45 minutes while carrying the heavy compute payload required for computer vision.

Utilizing the Jetson Orin Nano (8GB), the drone performs on-board image processing for crop health analysis and weather pattern recognition without reliance on ground station telemetry.

Mk II Systems Integration

  • Real-time NDVI mapping via Jetson AI edge computing.
  • High-efficiency power train using individual HolyBro Tekko32 F4 45A ESCs.
  • Redundant navigation via Matek H743 Wing V3 Flight Controller.
  • Carbon Fiber reinforced arms for increased rigidity.

Prototype Mk I

/// MK I LEGACY DATA

Initial Proof of Concept

The Mark I was the testbed for our initial propulsion theories. While it lacked the advanced AI processing capabilities of the Mark II, it provided crucial data regarding aerodynamic stress and battery thermal management.

This unit is no longer flight-capable but serves as a reference point for chassis structural integrity.

Mk I Key Learnings

  • Identified thermal throttling issues in standard ESCs.
  • Validated the vertical takeoff thrust-to-weight ratios.
  • Established the baseline for our telemetry and navigation software.
